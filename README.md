A simple ethereum wallet manipulation API.

To get started : 
  npm install && npm start
  

Options: 
  <ul>GET
    <li>/createWallet</li>
    <li>/getBalance:address</li>
  </ul>
  <ul>POST
    <li>/transaction</li>{privateKey, destination, amount}
  </ul>
